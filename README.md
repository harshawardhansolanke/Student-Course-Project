# Student-Course-Project-
This project demonstrates a basic SQL database system for managing students, courses, and their enrollments.
It includes table creation, data insertion, joins, aggregation.

# Project Overview-

1. students    –  stores student information
2. courses     –  stores available courses
3. enrollments –  links students to the courses they join

# Database Operations Included-

1. Creating tables using CREATE TABLE.
2. Adding constraints like PRIMARY KEY and FOREIGN KEY.
3. Inserting sample data.
4. Retrieving full table data.
5. Using JOIN to connect students with their courses.
6. Using GROUP BY to count how many students enrolled in each course.
7. Subquery to find the youngest student.
8. Final COMMIT to save data.

# Key SQL Queries-

1. Display all students and courses.
2. List each student with the course they enrolled.
3. Count enrollments per courses.
4. Find the youngest student in the database.
